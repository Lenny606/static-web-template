# Static Web Template

A reusable, mobile-first, SEO-friendly static website template.

## Features
- **No frameworks**: Pure HTML/CSS/JS.
- **Mobile-first**: Fluid design with clamp() typography and spacing.
- **Privacy-first**: GTM and tracking disabled by default (Consent Mode).
- **Performance**: High Lighthouse score ready.

## Directory Structure
- `/pages`: Content pages.
- `/assets`: CSS, JS, Images, Fonts.
- `index.html`: Homepage.

## Usage
1. Clone repository.
2. Edit `index.html` and pages in `/pages`.
3. Configure GTM ID in `assets/js/analytics.js` (to be created).
4. Deploy to any static host (Netlify, Vercel, FTP).
